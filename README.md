# VueShop
An attempt to explore the Vue framework and create a basic online shoe store again.

Technology stack for this project:

```
- Vue 3;
- Vue Router;
- TailwindCSS;
- Axios;
- @formkit/auto-animate;
- Composition API / Options API;
- ESLint;
- Prettier.
```
